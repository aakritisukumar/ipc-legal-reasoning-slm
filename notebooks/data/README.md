
# Dataset

## complete_ipc_dataset_500.json

500 manually curated examples of Indian Penal Code reasoning.

**Format:**
```json
{
  "id": 1,
  "question": "What is the punishment for murder under IPC Section 302?",
  "reasoning": "Step 1: Identify the provision...\nStep 2: ...",
  "answer": "Under Section 302 IPC, whoever commits murder..."
}
```

**Statistics:**
- Total examples: 500
- IPC sections covered: ~150
- Average question length: 14 words
- Average reasoning length: 195 words
- Average answer length: 142 words

**Coverage:**
Murder (302), dowry death (304B), theft (379), robbery (390-398), 
assault (323-326), kidnapping (363-369), and more.
